Resesif

Dark color scheme for Sublime Text
made by Carlos Cuesta

Resesif color scheme

Easy on the eyes, good contrast with nice colors.
Supports a lot of different languages.

RESESIF DEMO VIEW ON GITHUB

Install

Package Control

  1. Open command palette - ⇧ ⌘ P / ⇧ Ctrl P
  2. Install package ...
  3. Search for Resesif color scheme

Manual

  1. Locate your Sublime Text Packages directory
  2. Clone the repository

$ git clone https://github.com/carloscuesta/resesif.git

Use


  1. Sublime Text - Preferences - Color scheme
  2. Select Resesif
  3. Restart the text editor
